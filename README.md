**Syntax:**

There two ways to declare variable:
  - Keyword 'var':
```
var name = "John"
```
  - Character '_' as first symbol of variable name:
```
_age = 18
```
Two ways to declare function:
  - Keyword 'fun':
```
fun do_smth(arg1){
  println(arg1)
}

do_smth(name)
```
  - Character ':' as first symbol of function name:
```
:do_smth2(arg2){
  println(arg2)
}
do_smth2(age)
```
Built in functions:
  - *print()* - print text
`print(name)`
  - *println()* - print line of text
`println("Hello friday!")`
